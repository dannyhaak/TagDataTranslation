# TagDataTranslation MAUI Example

Minimal .NET MAUI application demonstrating EPC tag data translation on Android, iOS, and macCatalyst.

## Run

```bash
# macCatalyst
dotnet build examples/MauiApp/MauiApp.csproj -t:Run -f net10.0-maccatalyst

# Android (requires emulator)
dotnet build examples/MauiApp/MauiApp.csproj -t:Run -f net10.0-android
```
